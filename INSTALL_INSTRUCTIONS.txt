Problem: "pip is not recognized"
Solution: Make sure Python is installed and added to PATH
         Download from: https://www.python.org/downloads/

Problem: "python is not recognized"  
Solution: Try using "py" instead of "python"
         Example: py theta_wave_generator.py

Problem: Packages won't install
Solution: Try upgrading pip first:
         pip install --upgrade pip
         Then try installing packages again

Problem: Permission errors
Solution: Run Command Prompt as Administrator
         Right-click cmd.exe → "Run as administrator"
