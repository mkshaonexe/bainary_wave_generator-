   → Try running as Administrator (right-click → Run as admin)

Problem: No sound
   → Check headphones are plugged in
   → Check Windows volume settings
   → Try adjusting volume slider in the app

Problem: Antivirus blocks it
   → Add ThetaWaveGenerator.exe to your antivirus exceptions
   → The app is safe - this is a false positive
